The File System Access API and OPFS in modern Chromium

The File System Access API provides web applications a means of interacting with files on the user's local device. The Origin Private File System (OPFS) is a storage endpoint that is private to the origin and not exposed via the user-visible file system. It is optimized for performance and provides synchronous file access from within dedicated Web Workers.

OPFS files are stored in a sandboxed area, meaning they cannot be accessed by other origins, and users cannot directly view or edit them through the operating system's file picker. This makes OPFS well-suited for cases such as storing SQLite databases or large media buffers without polluting the user's downloads or document folders.
