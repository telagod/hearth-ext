01

四层切入 · 你不必再打开 app

L0

寄生候选

复制 / 深度阅读 / 截图 — 后台默默落到 Inbox，72 小时未确认自动消散。

L1

选中浮 bar

一秒按一下：📎 存 · 🤖 问 · 🔗 关联 · 💭 想。再也不必跳到主界面。

L2

反向召回 ☕

打开新页面时，旧笔记主动浮起。"Hearth 想起来了 — 你 3 周前读 X 时写过 Y。"

L3

周回顾

每周一 9 点，把上周高亮聚成 3-5 张主题卡。5 分钟刷一遍。

02

隐私 = 默认值

本地优先

所有笔记存在你这台设备的 OPFS（SQLite WAL）。永远不离开浏览器，除非你按下"问 AI"。

BYO API Key

Anthropic / OpenAI / Ollama 三选一，凭据存你本地。Hearth 永不持有，永不上报。

每次出网都审计

Settings → 过去 7 天云端调用，逐行可看：发了什么、几字节、几毫秒。

03

安装

1

克隆仓库

git clone <repo> hearth
cd hearth
npm install --legacy-peer-deps
2

构建

npm run build
3

加载

访问 chrome://extensions → 开发者模式 → 加载已解压 → 选 dist/

4

配 LLM (可选)

点工具栏 ☕ 图标 → Settings → Provider → 填 API key → 点「我同意」

Chrome Web Store 上架包准备中 · 关注 GitHub 获取更新
04
